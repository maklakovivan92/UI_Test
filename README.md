# Проект содержит автотесты на Python + Selenium для сайта  
# https://stellarburgers.education-services.ru/

# Тесты проверяют основные пользовательские сценарии:
# - авторизацию
# - регистрацию
# - работу личного кабинета
# - навигацию по разделам конструктора бургера


## Покрытие тестами

# В проекте реализовано **13 проверок**:

### Авторизация (4 проверки)

# 1. Вход по кнопке **«Войти в аккаунт»** на главной странице  
# 2. Вход через кнопку **«Личный кабинет»**  
# 3. Вход через кнопку в форме **регистрации**  
# 4. Вход через кнопку в форме **восстановления пароля**

### Регистрация (2 проверки)

# 5. Регистрация с **валидными данными**  
# 6. Регистрация с **невалидной длиной пароля**

### Выход из аккаунта

# 7. Выход из личного кабинета

### Личный кабинет

# 8. Переход из конструктора по клику на **«Личный кабинет»**  
# 9. Переход из личного кабинета по клику на логотип **Stellar Burgers** в конструктор

### Конструктор бургеров

# 10. Переход к разделу **«Начинки»**  
# 11. Переход к разделу **«Соусы»**  
# 12. Переход к разделу **«Булки»**
